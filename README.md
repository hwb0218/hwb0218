# 🐮 한우빈 
```
재미있고 의미 있는 서비스를 만들어 행복의 가치를 나누고 싶은 프론트엔드 개발자 한우빈입니다.
```

</br>

## 👨‍💻 Introduce 

행복의 핵심 가치는 재미와 의미라고 합니다.

재미있고 의미 있는 서비스를 만들어 고객에게 행복을 전해주고 싶습니다.

장래희망은 내가 배운 지식을 나누어 주는 지식전도사입니다.
